## 基础数据类型
  - undefined number string boolean fucntion object symbol
## 对象类型 
  - object function array regexp
## 显示包装
在js中内建的Object()类支持显示地将boolean number string symbol四种值类型数据包装成对应的对象

例如:
```javascript
  new Object(1)
  new Object('a')
  new Object(true)
```
### 隐式包装
对于值类型来说是发生于对该值类型进行`成员存取`运算触发的，普通的赋值以及运算不会触发隐式包装的！
